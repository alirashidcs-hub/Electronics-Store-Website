# Electronics Store Website

## Overview

Electronics Store Website is a front-end web development project created as part of my Information and Communication Technology (ICT) Lab course. The project simulates an online electronics shopping platform with a clean, responsive, and user-friendly interface.

The website demonstrates the fundamentals of web development, including webpage structure, navigation, content organization, and responsive design using HTML.

- Product Categories
